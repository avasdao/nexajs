# NEXA.js Application Creator

Quickly and easily design and launch your next __Unstoppable Web__ application.

```shell
npm create nexa
```

```shell
yarn create nexa
```

```shell
pnpm create nexa
```

#### To update to the LATEST version, please run:
```shell
npx create-nexa@latest
```
